Student Score Manager（Python練習プロジェクト）


概要
このプログラムは、複数人の名前と点数を入力し、
合否判定・集計・結果表示を行うシンプルなPython CLIアプリ。

Pythonの基礎学習として以下を練習する目的で作成した。

- 入力処理（input）
- 条件分岐（if）
- ループ（for / while）
- 関数（return）
-  リスト・辞書の操作
-  例外処理（try / except）


機能

複数人の名前と点数を入力
点数のバリデーション（0〜100）
合否判定（60点以上で合格）
結果一覧の表示
合格・不合格人数の集計
平均点の計算
